# 科学技術計算講座５－粒子法（SPH法）で流体シミュレーション
## 概要
「科学技術計算講座５」のサンプルプログラムです。  
https://cattech-lab.com/science-tools/simulation-lecture/#lec5

２次元の粒子法（SPH法）のJavaScriptプログラムです。
